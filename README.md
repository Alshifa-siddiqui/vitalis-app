# 🌿 Vitalis

A secure, cross-platform **habit & wellness app** built with Expo (React Native) + TypeScript and a Supabase backend. Track habits, build streaks, earn badges, and get AI-powered wellness insights — with email/Google auth, cloud sync, dark mode, and native reminders.

## ✨ Features
- **Auth** — email/password + Google sign-in (Supabase), encrypted token storage (Keychain/Keystore)
- **Cloud sync** — habits sync to Postgres with Row-Level Security (per-user isolation)
- **Habits** — create/edit/delete, daily/weekly/monthly, categories, derived streaks & badges
- **AI insights** — goal-aware wellness coaching via Claude, called from a secure Supabase Edge Function (API key never ships in the app)
- **Reminders** — daily local notifications per habit
- **Onboarding** — welcome flow + goal selection that seeds starter habits and tailors AI
- **Dark mode**, brand fonts, animated lotus, empty states, pull-to-refresh
- **Profile** — editable profile, health profile, notification + privacy/data controls

## 🧱 Tech stack
Expo SDK · React Native · TypeScript · Zustand (+ AsyncStorage persistence) · Supabase (Postgres, Auth, Edge Functions) · react-native-svg · Anthropic Claude API (server-side).

## 🚀 Getting started
```bash
npm install
npm run web        # or: npx expo start  (then Expo Go / a dev build)
```

### Environment
Copy `.env.example` → `.env` and fill in your Supabase project values:
```
EXPO_PUBLIC_SUPABASE_URL=https://YOUR_PROJECT.supabase.co
EXPO_PUBLIC_SUPABASE_ANON_KEY=YOUR_ANON_PUBLIC_KEY
```
The app runs in a local "demo mode" until these are set.

### Backend setup
1. Create a Supabase project.
2. SQL Editor → run `supabase/schema.sql` (creates the `habits` table + RLS).
3. Enable the Google provider (Authentication → Providers) if you want Google sign-in.

### AI Edge Function
```bash
npx supabase functions deploy ai-insight --project-ref <your-ref>
npx supabase secrets set ANTHROPIC_API_KEY=sk-ant-...   # your Anthropic key
```

## 📁 Structure
```
App.tsx                 app shell, auth/onboarding gate, tab nav
src/
  theme.ts useColors.ts light/dark palettes + hook
  store.ts              Zustand store (persisted)
  streaks.ts            derived streak/badge logic (ported + unit-tested)
  supabase.ts auth.tsx sync.ts ai.ts reminders.ts
  ui.tsx Lotus.tsx      shared components
  screens/              Home, Habits, Health, Progress, Profile, Auth, Onboarding
supabase/
  schema.sql            DB schema + Row-Level Security
  functions/ai-insight  Claude Edge Function
assets/                 app icon, adaptive icon, splash (generated by scripts/gen-icons.mjs)
```
